# specs — repo-wide

Formal cross-module contracts and lesson-level specs (L01–L08). Put a doc
here when more than one module needs to agree on it. Single-module specs
live in `<module>/specs/`.

Suggested layout:

- `L0N-<slug>.md` — lesson specification
- `contracts/<name>.md` — cross-module contract (e.g. SSE event shape,
  finding lifecycle)
- `rfc/NNNN-<slug>.md` — proposals before implementation
